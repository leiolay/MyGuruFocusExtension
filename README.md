# GuruFocus Quick Search

Select a ticker symbol and open the stock summary page on gurufocus. Use Alt+Shift+G for a new tab or Alt+Shift+B for a background tab.

**Note:** Opening the tab in the background only works in Chrome, but not in ChatGPT Atlas.

## Installation

### Chrome

1. Download the extension files.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable "Developer mode" in the top right corner.
4. Click "Load unpacked" and select the directory where you downloaded the extension.

### ChatGPT Atlas

1. Download the extension files.
2. Open ChatGPT Atlas and navigate to `atlas://extensions`.
3. Enable "Developer mode" in the top right corner.
4. Click "Load unpacked" and select the directory where you downloaded the extension.
